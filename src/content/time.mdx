## 🔁 核心时间单位换算公式

> 💡 注：为简化计算，本工具对“月”和“年”采用**平均值**：
> - 1 年 = 365.25 天（考虑闰年）
> - 1 月 = 365.25 ÷ 12 ≈ 30.4375 天

### 精确换算关系（基于秒）

| 转换 | 公式 |
|------|------|
| ns → mu | mu = ns ÷ 1,000 |
| mu → ms | ms = mu ÷ 1,000 |
| ms → s  | s = ms ÷ 1,000 |
| s → min | min = s ÷ 60 |
| min → h | h = min ÷ 60 |
| h → d   | d = h ÷ 24 |
| d → week| week = d ÷ 7 |
| d → month| month = d ÷ 30.4375 |
| d → year| year = d ÷ 365.25 |

### 基准换算（以 1 秒为单位）

| 单位 | 等于多少秒 |
|------|------------|
| 1 ns    | 1e-9 s     |
| 1 mu (μs)| 1e-6 s     |
| 1 ms    | 0.001 s    |
| 1 s     | 1 s        |
| 1 min   | 60 s       |
| 1 h     | 3,600 s    |
| 1 d     | 86,400 s   |
| 1 week  | 604,800 s  |
| 1 month | ≈ 2,629,746 s |
| 1 year  | ≈ 31,557,600 s |
