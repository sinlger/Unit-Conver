## 🔁 核心面积单位换算公式

### 公制单位之间（基于平方米）

| 转换 | 公式 |
|------|------|
| mm² → cm² | cm² = mm² ÷ 100 |
| cm² → m²  | m² = cm² ÷ 10,000 |
| m² → ha   | ha = m² ÷ 10,000 |
| ha → km²  | km² = ha ÷ 100 |
| m² → km²  | km² = m² ÷ 1,000,000 |

### 英制单位之间

| 转换 | 公式 |
|------|------|
| in² → ft² | ft² = in² ÷ 144 |
| ft² → yd² | yd² = ft² ÷ 9 |
| ac → mi²  | mi² = ac ÷ 640 |
| yd² → ac  | ac = yd² ÷ 4,840 |

### 公制 ↔ 英制 关键换算基准

| 基准关系 | 精确值 |
|----------|--------|
| 1 in²  | 6.4516 cm² |
| 1 ft²  | 0.092903 m² |
| 1 yd²  | 0.836127 m² |
| 1 ac   | 4,046.85642 m² |
| 1 mi²  | 2.58998811 km² |

> 💡 小知识：  
> - **1 公顷（ha） = 10,000 m²**，约等于一个标准足球场大小。  
> - **1 英亩（ac） ≈ 0.4047 公顷**，常用于美国、英国的土地面积描述。
