## 🔁 核心换算公式

### 公制单位（基于立方米 m³）

| 转换 | 公式 |
|------|------|
| mm³ → cm³ | cm³ = mm³ ÷ 1,000 |
| cm³ → ml  | 1 cm³ = 1 ml（等价） |
| ml → l    | l = ml ÷ 1,000 |
| l → kl    | kl = l ÷ 1,000 |
| l → m³    | m³ = l ÷ 1,000 |
| m³ → km³  | km³ = m³ ÷ 1,000,000,000 |

> 💡 提示：1 cm³ = 1 ml，这是连接体积与容积的关键桥梁。

### 升系列细分（欧洲常用）

| 单位 | 等于多少毫升（ml） |

| 1 tsk（茶匙）| = 5 ml |
| 1 msk（汤匙）| = 15 ml |

| 1 kanna     | ≈ 2,000 ml = 2 l（瑞典传统壶）|

> ⚠️ 注意：北欧厨房单位因国家略有差异，本工具采用**瑞典标准**（最常用）。
