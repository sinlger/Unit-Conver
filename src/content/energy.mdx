## 🔁 核心能量单位换算公式

### 基础关系：瓦时 与 焦耳 的换算

> ⚡ **1 瓦时（Wh） = 3,600 焦耳（J）**  
> （因为 1 W = 1 J/s，1 小时 = 3,600 秒 → 1 Wh = 1 J/s × 3,600 s = 3,600 J）

由此可推导所有换算：

| 转换方向 | 公式 |
|----------|------|
| Wh → J   | J = Wh × 3,600 |
| J → Wh   | Wh = J ÷ 3,600 |
| kWh → J  | J = kWh × 3,600,000 |
| J → kJ   | kJ = J ÷ 1,000 |
| mWh → Wh | Wh = mWh ÷ 1,000 |
| Wh → kWh | kWh = Wh ÷ 1,000 |
| kWh → MWh| MWh = kWh ÷ 1,000 |
| MWh → GWh| GWh = MWh ÷ 1,000 |
| GWh → J  | J = GWh × 3.6e12 |
