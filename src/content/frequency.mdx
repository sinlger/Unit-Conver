# 频率与角速度单位转换工具

在电子工程、通信技术、机械设计、物理实验乃至音频处理中，**频率**和**角速度**的单位转换是基础而关键的操作。然而，不同领域使用不同的单位体系：

- **频率单位**：mHz、Hz、kHz、MHz、GHz、THz（用于信号、时钟、无线电波等）

## 🔁 核心换算公式

### 1. 频率单位（基于赫兹 Hz，十进制）

| 转换 | 公式 |
|------|------|
| mHz → Hz   | Hz = mHz ÷ 1,000 |
| Hz → kHz   | kHz = Hz ÷ 1,000 |
| kHz → MHz  | MHz = kHz ÷ 1,000 |
| MHz → GHz  | GHz = MHz ÷ 1,000 |
| GHz → THz  | THz = GHz ÷ 1,000 |

> 💡 1 Hz = 1 次/秒（cycles per second）

### 2. 角速度与频率的相互转换

一个完整旋转 = 360° = 2π 弧度 ≈ 6.28319 rad

| 转换方向 | 公式 |
|----------|------|
| rpm → Hz     | Hz = rpm ÷ 60 |
| Hz → rpm     | rpm = Hz × 60 |
| rpm → rad/s  | rad/s = rpm × (2π ÷ 60) ≈ rpm × 0.10472 |
| rad/s → rpm  | rpm = rad/s × (60 ÷ 2π) ≈ rad/s × 9.5493 |
| deg/s → rad/s| rad/s = deg/s × (π ÷ 180) ≈ deg/s × 0.0174533 |
| rad/s → deg/s| deg/s = rad/s × (180 ÷ π) ≈ rad/s × 57.2958 |
| Hz → rad/s   | rad/s = Hz × 2π ≈ Hz × 6.28319 |
| rad/s → Hz   | Hz = rad/s ÷ 2π ≈ rad/s × 0.159155 |
