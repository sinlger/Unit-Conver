# 角度单位转换工具

在数学、物理、工程制图、天文学、地理测绘乃至游戏开发中，**角度的表示方式多种多样**。不同领域偏好不同的单位：

- **弧度（rad）**：国际单位制（SI）标准，用于高等数学与物理计算  
- **度（deg）**：最广为人知的角度单位，日常和基础教育常用  
- **梯度（grad）**：又名“百分度”，在部分欧洲国家和土木工程中使用  
- **角分（arcmin）与角秒（arcsec）**：天文学、导航、光学中用于表示微小角度  

手动在这些单位间换算不仅繁琐，还容易因 π 或 60 进制出错。

## 🔁 核心角度换算公式

### 基准关系（基于一个完整圆周 = 360°）

| 单位 | 等于多少度（°） |
|------|----------------|
| 1 rad       | ≈ 57.295779513° |
| 1 grad      | = 0.9°          |
| 1 arcmin    | = 1/60° ≈ 0.0166667° |
| 1 arcsec    | = 1/3600° ≈ 0.000277778° |

> 💡 关键常数：  
> - **π 弧度 = 180°** → 1 rad = 180/π ≈ 57.2958°  
> - **1 圆周 = 400 grad**（梯度制将直角定义为 100 grad）

### 常用转换公式

| 转换方向 | 公式 |
|----------|------|
| deg → rad    | rad = deg × π / 180 |
| rad → deg    | deg = rad × 180 / π |
| deg → grad   | grad = deg × 10/9 |
| grad → deg   | deg = grad × 0.9 |
| deg → arcmin | arcmin = deg × 60 |
| arcmin → deg | deg = arcmin ÷ 60 |
| arcmin → arcsec | arcsec = arcmin × 60 |
| deg → arcsec | arcsec = deg × 3,600 |
| rad → arcsec | arcsec = rad × (180/π) × 3,600 ≈ rad × 206,264.8 |
